## 2. What is EZTOLLER Options

EZTOLLER Options is a software designed by DAOdesign team, which can easily adjust the parameters of EZTOLLER5 and EZTOLLER9. You can adjust the parameters to make EZTOLLER5 and EZTOLLER9 more comfortable to use.

## 3. How to Use

### 3.1 How to Connect

1. Connect the USB type-b cable, one end to EZTOLLER5 or EZTOLLER9, and the other end to the computer USB port (USB 3.0 port is recommended).
2. The Type-c port can be connected or not.
3. Download the latest version of EZTOLLER Options.
4. Unzip the package, which contains the following files:
   - `languagepack.xml` (the language file)
   - `EZTOLLER-Options-XXX.exe` (the software body)
   - `SS900_EZ5_APP_RC_VXXX.skystar` (the factory firmware)

### 3.2 How to Burn the Firmware

1. Click the "Open" button;
2. Find the `SS900_EZ5_APP_RC_VXXX.bin` file in the directory;
3. Click the "Upgrade" button;

### 3.5 Keyboard Mode

![Keyboard Mode](/img/docs/eztoller5/options-tutorial/en/keyboard-mode.png)

| No. | Function Description |
|-----|---------------------|
| 1 | Multiple keyboard values can be mapped to one button. (A single button supports binding 1, 2, 3, or 4 keyboard values at the same time) |
| 2 | Click the button and a pop-up window will appear, where you can bind and clear the key value |
| 3 | Map the key value of the turntable clockwise |
| 4 | Map the key value of the turntable counterclockwise |
| 5 | The larger the value, the larger the angle of the turntable needs to be rotated |
| 6 | Decide whether to increase it based on the game characteristics. Judgment basis: If the turntable is rotated back and forth at a high frequency, if there are missed keys, it is recommended to increase it |
| 7 | The larger the value, the turntable can also output the key value when it rotates slowly. The smaller the value, the turntable stops outputting the key value when it is lower than a certain speed. For Longnote, you can make fine adjustments based on the characteristics |
| 8 | Restore the current mode to the factory state |

### 3.6 Keyboard ON Mode

![Keyboard ON Mode](/img/docs/eztoller5/options-tutorial/en/keyboard-on-mode.png)

| No. | Function Description |
|-----|---------------------|
| 1 | Clockwise and Counterclockwise can map the same key value |
| 2 | According to the game characteristics of EZ2ON, if the turntable output is mapped to two same keyboard value, an 8ms delay is required. If EZ2ON is upgraded in the future, more delay may not be needed here |
| 3 | Restore the current mode to the factory state |
| PS | Keyboard ON and Keyboard mode are same. The two modes can be used as the default input for different games |

### 3.7 K&S Mode (Keyboard & Mouse)

![K&S Mode](/img/docs/eztoller5/options-tutorial/en/ks-mode.png)

| No. | Function Description |
|-----|---------------------|
| 1 | The turntable is mapped to Mouse X-axis or Y-axis |
| 2 | Output mouse displacement is in the opposite direction |
| 3 | Preview output effect |
| 4 | The larger the value, the larger the angle of the turntable needs to be rotated |
| 5 | 1:1, one turn, output 288 mouse movements; 1:2, one turn, output 144 mouse movements; 1:4, one turn, output 72 mouse movements |
| 6 | The larger the value, the mouse value can be output when the turntable rotates slowly; The smaller the value, the mouse value will stop being output when the turntable is below a certain speed; For Longnote, fine-tuning can be done according to the characteristics |
| 7 | If there is no special situation, use 1; If there is high-frequency back and forth, the output is missed, it can be changed to a larger value |
