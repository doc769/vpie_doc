# GITALLER switch shaft circuit board replacement tutorial

Difficulty: 2/5 | Estimated time: about 15-20 minutes

## Shaft circuit board types

| Circuit board | Suitable for |
|:---:|:---:|
| Golden switch shaft circuit board | 3-finger players |
| Brown switch shaft circuit board | 4-finger players |

## Replacement tutorial

<Steps>
<Step title="Prepare tools">

Prepare Phillips screwdriver (preferably magnetic) + 2.5mm hexagon socket screwdriver.

![Required tools](/img/docs/gitaller/replace-switch-pcb/step01-tools.png)

</Step>
<Step title="Loosen strap screws">

Loosen 2 screws at the strap position with 2.5mm hex driver (just loosen, do not remove).

![Loosen strap screws 1](/img/docs/gitaller/replace-switch-pcb/step02-strap-screws-1.png)

![Loosen strap screws 2](/img/docs/gitaller/replace-switch-pcb/step02-strap-screws-2.png)

</Step>
<Step title="Unscrew in order">

Unscrew screws in the order shown.

![Unscrew order](/img/docs/gitaller/replace-switch-pcb/step03-unscrew-order.png)

</Step>
<Step title="Unscrew 10 screws">

Unscrew 10 screws.

![Unscrew 10 screws](/img/docs/gitaller/replace-switch-pcb/step04-10-screws.png)

</Step>
<Step title="Open bottom cover">

Open from the upper part, unplug the 5PIN USB cable, loosen the USB cable clip.

![Open cover 1](/img/docs/gitaller/replace-switch-pcb/step05-open-cover-1.png)

![Unplug USB](/img/docs/gitaller/replace-switch-pcb/step05-open-cover-2.png)

![Loosen clip](/img/docs/gitaller/replace-switch-pcb/step05-open-cover-3.png)

</Step>
<Step title="Remove screws in order">

Remove screws in the order shown.

![Remove screws 1](/img/docs/gitaller/replace-switch-pcb/step06-remove-screws-1.png)

![Remove screws 2](/img/docs/gitaller/replace-switch-pcb/step06-remove-screws-2.png)

</Step>
<Step title="Unscrew gyro PCB">

Unscrew 2 screws of the gyro PCB.

![Unscrew gyro PCB](/img/docs/gitaller/replace-switch-pcb/step07-gyro-pcb.png)

</Step>
<Step title="Unscrew shaft PCB">

Unscrew 4 screws of the shaft PCB.

![Unscrew shaft PCB](/img/docs/gitaller/replace-switch-pcb/step08-shaft-pcb.png)

</Step>
<Step title="Take out shaft PCB">

Take out the shaft PCB and remove the 6PIN wire (push upward).

![Take out PCB](/img/docs/gitaller/replace-switch-pcb/step09-remove-6pin-1.png)

![Remove 6PIN wire](/img/docs/gitaller/replace-switch-pcb/step09-remove-6pin-2.png)

</Step>
<Step title="Remove keycaps">

Remove all keycaps.

![Remove keycaps](/img/docs/gitaller/replace-switch-pcb/step10-remove-keycaps.png)

</Step>
<Step title="Remove limit blocks">

Remove 10 limit blocks (push finger down about 1mm gently, push laterally).

![Limit block removal 1](/img/docs/gitaller/replace-switch-pcb/step11-limit-blocks-1.png)

![Limit block removal 2](/img/docs/gitaller/replace-switch-pcb/step11-limit-blocks-2.png)

![Successfully removed](/img/docs/gitaller/replace-switch-pcb/step11-limit-blocks-3.png)

![All removed](/img/docs/gitaller/replace-switch-pcb/step11-limit-blocks-4.png)

</Step>
<Step title="Install keycaps to new PCB">

Install keycaps to the gold switch shaft PCB, check that all 5 keycaps are the same height.

![Install keycaps](/img/docs/gitaller/replace-switch-pcb/step12-install-keycaps-1.png)

![Check height](/img/docs/gitaller/replace-switch-pcb/step12-install-keycaps-2.png)

</Step>
<Step title="Install limit blocks">

Install limit blocks (pay attention to direction).

![Block direction](/img/docs/gitaller/replace-switch-pcb/step13-install-blocks-1.png)

![1 installed](/img/docs/gitaller/replace-switch-pcb/step13-install-blocks-2.png)

![10 installed](/img/docs/gitaller/replace-switch-pcb/step13-install-blocks-3.png)

</Step>
<Step title="Install gold switch shaft PCB">

Install the gold switch shaft PCB (leave 0.2mm gap for fine adjustment).

![Leave gap](/img/docs/gitaller/replace-switch-pcb/step14-install-pcb-1.png)

![Install PCB](/img/docs/gitaller/replace-switch-pcb/step14-install-pcb-2.png)

</Step>
<Step title="Install wire harness">

Remove screws from the original PCB, install on new PCB.

![Remove screws](/img/docs/gitaller/replace-switch-pcb/step15-wire-harness-1.png)

![Install on new PCB](/img/docs/gitaller/replace-switch-pcb/step15-wire-harness-2.png)

</Step>
<Step title="Install gyroscope PCB">

Install the gyroscope PCB.

![Install gyro PCB](/img/docs/gitaller/replace-switch-pcb/step16-gyro-install.png)

</Step>
<Step title="Fix wire position">

Use wire harness to fix wire position (leave gaps between wire and DIP pad).

![Fix wire 1](/img/docs/gitaller/replace-switch-pcb/step17-fix-wire-1.png)

![Fix wire 2](/img/docs/gitaller/replace-switch-pcb/step17-fix-wire-2.png)

</Step>
<Step title="Install screws">

Install screws (lift guitar neck gently to make it easier).

![Lift neck](/img/docs/gitaller/replace-switch-pcb/step18-install-screws-1.png)

![Tighten in order](/img/docs/gitaller/replace-switch-pcb/step18-install-screws-2.png)

</Step>
<Step title="Handle button wires">

Handle Select/Start button wires as shown.

![Handle button wires](/img/docs/gitaller/replace-switch-pcb/step19-button-wire.png)

</Step>
<Step title="Handle bottom cover">

Insert 5PIN USB, arrange wire harness.

![Insert USB](/img/docs/gitaller/replace-switch-pcb/step20-bottom-cover-1.png)

![Arrange harness](/img/docs/gitaller/replace-switch-pcb/step20-bottom-cover-2.png)

![Close cover](/img/docs/gitaller/replace-switch-pcb/step20-bottom-cover-3.png)

</Step>
<Step title="Install bottom cover screws">

Bottom cover screws in order (screws 3 & 4 should not be tightened too much).

![Screw order 1](/img/docs/gitaller/replace-switch-pcb/step21-cover-screws-1.png)

![Screw order 2](/img/docs/gitaller/replace-switch-pcb/step21-cover-screws-2.png)

![Screw order 3](/img/docs/gitaller/replace-switch-pcb/step21-cover-screws-3.png)

![Screw order 4](/img/docs/gitaller/replace-switch-pcb/step21-cover-screws-4.png)

</Step>
<Step title="Fine-tune (if needed)">

If button friction is too large: keep gaps uniform, adjust shaft PCB (0.2mm in 4 directions).

![Keep gaps uniform](/img/docs/gitaller/replace-switch-pcb/step22-adjust-1.png)

![Fine-tune 0.2mm](/img/docs/gitaller/replace-switch-pcb/step22-adjust-2.png)

</Step>
<Step title="Done">

Replacement complete, test all buttons to ensure they work properly.

</Step>
</Steps>

If you cannot resolve the issue, please email us.
:::
