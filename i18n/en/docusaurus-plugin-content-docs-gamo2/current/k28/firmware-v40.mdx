# K28 firmware update V40 20220628

## 1 New Function

This firmware contains all the functions of firmware V39.

1. Clear the bug of multiple button reactions when using the 3.5mm audio interface in keyboard mode (B11, B12 layout).
2. In keyboard mode (B11, B12 layout), optimize the effective reporting rate, which can be maintained at about 1ms continuously.

## 3 Firmware Burn Tutorial

### 3.1 HOST Firmware

1. Unplug the K28 USB-C cable, press **B42 + B44** without letting go, insert the USB-C port, and release B42 and B44 after **2 seconds**.
2. Unzip the firmware archive. Open the **Update V1.1.exe** file and click Connect. If the Connect button turns green, the connection is successful. Otherwise, repeat action 1.
3. Click the Open files button and select the firmware file "**HOST_xxxx.bin**" to be upgraded.
4. Click the Update button.
5. Wait for the progress bar to complete, then **replug the K28 USB-C cable**.
6. Update completed.

### 3.2 LED Firmware

1. Unplug the K28 USB-C cable, press **B41 + B43** without letting go, insert the USB-C port, and release B41 and B43 after **5 seconds**.
2. Unzip the firmware archive. Open the **Update V1.1.exe** file and click Connect. If the Connect button turns green, the connection is successful. Otherwise, repeat action 1.
3. Click the Open files button and select the firmware file "**LED_xxxx.bin**" to be upgraded.
4. Click the Update button.
5. Wait for the progress bar to complete, then **replug the K28 USB-C cable**.
6. Update completed.


## 4 Q&A

<FaqAccordion>
<FaqItem question="What should I pay attention to when updating the firmware?">

After pressing the key combination, plug in the USB port. After inserting the USB, hold it for about 2 seconds, and then release the key combination. Windows needs some time to react. Otherwise, the K28 device cannot be connected.

</FaqItem>
<FaqItem question="Can I skip the previous firmware and directly upgrade this firmware?">

Sure. This firmware has all the functions of the previous firmware.

</FaqItem>
<FaqItem question="What should I do if the firmware update prompts an error?">

The data transmission error may be caused by the poor contact of the USB interface. Just update the firmware again.

</FaqItem>
<FaqItem question="I don't like the current firmware, can I flash back to the previous version?">

Of course, click to download the firmware, 20200221.zip

</FaqItem>
<FaqItem question="How to customize my own layout?">

Please read the manual carefully. Usually, we recommend: ①Draw the layout you want on white paper first; ②Use the key combination to enter the number of the layout you want to customize; ③Set the key value of each button; ④Save.

</FaqItem>
<FaqItem question="Why is it prompting invalid when I click Connect?">

Please do not delete the **Config.ini** file. If it is accidentally deleted, please download the firmware upgrade package again and unzip it again.

</FaqItem>
<FaqItem question="On Steam, how do I use the PlayStation layout?">

If the game supports PlayStation controllers, please use any layout of the K28 Steam (B44+B13, B44+B14).

</FaqItem>
