# Contribute to Docs

Thank you for your interest in contributing to nanos world documentation!

## How to Contribute

1. Fork the documentation repository
2. Create your branch
3. Submit your changes
4. Open a Pull Request

## Translation Contributions

We use Crowdin for documentation translations. Feel free to help us translate the documentation into more languages.
