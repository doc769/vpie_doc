## ファームウェアアップグレードチュートリアル

<Steps>
<Step title="ファームウェアの圧縮パッケージを解凍">

ダウンロードしたファームウェアの圧縮パッケージをローカルフォルダーに解凍します。

</Step>
<Step title="USBが未接続であることを確認">

GITALLER の USB ポートが PC に接続されていないことを確認します。

</Step>
<Step title="ファームウェアアップグレードモードに入る">

手を放さずに START + SELECT ボタンを同時に押します。GITALLER の USB ポートを PC に挿入します。USB を PC に接続したら、START + SELECT ボタンを放してください。

![START + SELECTを押す](/img/docs/gitaller/firmware-20200926/step3-buttons.png)

![USBを挿入](/img/docs/gitaller/firmware-20200926/step3-usb.png)

</Step>
<Step title="ファームウェアアップグレードプログラムを開く">

`GITALLER_firmware_APP.exe` プログラムを開きます。

![プログラムを開く](/img/docs/gitaller/firmware-20200926/step4-app.png)

</Step>
<Step title="ファームウェアファイルを読み込む">

**LoadFile** ボタンをクリックし、ダウンロードしたファームウェアフォルダーを見つけて、`GITALLER_firmware_V1.0_20200926.bin` を選択します。

![LoadFileをクリック](/img/docs/gitaller/firmware-20200926/step5-loadfile.png)

![ファームウェアファイルを選択](/img/docs/gitaller/firmware-20200926/step6-select-bin.png)

</Step>
<Step title="アップグレード開始">

**Start** ボタンをクリックして、ファームウェアのアップグレードを開始します。進行状況バーが完了するのを待ちます。

![Startをクリック](/img/docs/gitaller/firmware-20200926/step7-start.png)

![進行状況バーを待つ](/img/docs/gitaller/firmware-20200926/step8-progress.png)

</Step>
<Step title="アップグレード完了">

成功後、**GITALLER USB インターフェースを再接続します**（重要）。

![アップグレード成功](/img/docs/gitaller/firmware-20200926/step9-success.png)

:::warning
成功しない場合は、ステップ 3 から操作を再度行ってください。
:::

</Step>
</Steps>

## ファームウェアアップグレード後の機能紹介

### WAILING ボタンモード

このモードはデフォルトであり、切り替える必要はありません。

B1+B2+B3+B4+B5+SELECT を 2 秒間同時に押して、ジャイロスコープの XYZ 軸出力をシールドし、wailing_up と wailing_down をボタンとして出力します。

![WAILINGボタンモード](/img/docs/gitaller/firmware-20200926/wailing-button-mode.png)

### WAILING XYZ モード

このモードは上級者向けです。初心者は切り替えないでください。

B1+B2+B3+B4+B5+START を 2 秒間同時に押して、wailing_up と wailing_down のボタン出力をシールドし、ジャイロスコープの XYZ 軸が正常に出力されます。

![WAILING XYZモード](/img/docs/gitaller/firmware-20200926/wailing-xyz-mode.png)

## 解決された問題

1. GITADORA trial 1.0 において、START ボタンは START 機能、SELECT ボタンは HELP 機能です
2. DTXmania の設定時は、WAILING BUTTON モードを使用してください。wailing_up と wailing_down の 2 つのボタンをマッピングする必要があります
3. S tools や B tools を使用する場合：XYZ 軸をマッピングする場合は WAILING XYZ モードに切り替えてください。wailing_up と wailing_down ボタンをマッピングする場合は WAILING BUTTON モードに切り替えてください

### GITADORA trial の設定図

GITADORA trial では、第 1 ボタンは 0、wailing_up は 10、wailing_down は 11 と表示されます。

![GITADORA trial設定](/img/docs/gitaller/firmware-20200926/gitadora-trial-config.png)

### DTXmania の設定図

DTXmania では、第 1 ボタンは 1、wailing_up は 11、wailing_down は 12 と表示されます。

![DTXmania設定](/img/docs/gitaller/firmware-20200926/dtxmania-config.png)
