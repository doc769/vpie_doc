# TAIKOLLER 分解手順

<Steps>

<Step title="鼓面を外す">

画像の様に、縁を隙間から一周巡って、鼓面を外す。

![鼓面を外す](/img/docs/taikoller/disassembly/step1.png)

</Step>

<Step title="スタンドを取り外す">

逆様に置いて、マーキングされた位置のネジを手で外し、スタンドを取り外す。

![スタンドを取り外す](/img/docs/taikoller/disassembly/step2.png)

</Step>

<Step title="筐体のボルトを外す">

コントローラを内蔵する部分の蓋を取って、付属のドライバーで筐体のボルトを外す。

![筐体のボルトを外す](/img/docs/taikoller/disassembly/step3.png)

</Step>

<Step title="筐体を開ける">

筐体を横側に置いて、コネクターを包んでいる黒い緩衝材を固定する金属バンドを解く。

![筐体を開ける 1](/img/docs/taikoller/disassembly/step4-1.png)

![筐体を開ける 2](/img/docs/taikoller/disassembly/step4-2.png)

</Step>

<Step title="センサーのコネクターを外す">

センサーのコネクターを外す。

![センサーのコネクターを外す 1](/img/docs/taikoller/disassembly/step5-1.png)

![センサーのコネクターを外す 2](/img/docs/taikoller/disassembly/step5-2.png)

![センサーのコネクターを外す 3](/img/docs/taikoller/disassembly/step5-3.png)

</Step>

<Step title="金属バンドを解く">

金属バンドを解いて、配線を取り出す。

![金属バンドを解く](/img/docs/taikoller/disassembly/step6.png)

</Step>

<Step title="後側のナットを取り外す">

M6（H10）レンチで後側のナットを取り外す。

:::danger 重要
ナットの位置は決められたもので、戻す時はそれぞれ元の位置に戻さなければなりませんので、外した後、乱れない様に、順番で、ナットを横においてください。
:::

![後側のナットを取り外す 1](/img/docs/taikoller/disassembly/step7-1.png)

![後側のナットを取り外す 2](/img/docs/taikoller/disassembly/step7-2.png)

</Step>

<Step title="板を取り出す">

センサーを付けている板を取り出す際は、画像の様にコネクターをそれぞれの穴に入れてから、板を取り出す。配線を強く引っ張らない様ご注意ください。

:::warning 注意
- 複数の板を同時に外すと、戻す時に難しくなるので、ご注意ください。
- 板を取り出す際に、横にあるナットにぶつからない様ご注意ください。
:::

![板を取り出す 1](/img/docs/taikoller/disassembly/step8-1.png)

![板を取り出す 2](/img/docs/taikoller/disassembly/step8-2.png)

</Step>

<Step title="センサーを外す">

センサーを固定する金具を外してから、センサーを外す。

:::warning 注意
横にあるナットにぶつからない様ご注意ください。
:::

![センサーを外す 1](/img/docs/taikoller/disassembly/step9-1.png)

![センサーを外す 2](/img/docs/taikoller/disassembly/step9-2.png)

</Step>

<Step title="ナットを戻す">

ナットを戻す際に、画像の様に、マークがつながっているのをご確認ください。

![ナットを戻す](/img/docs/taikoller/disassembly/step10.png)

</Step>

<Step title="筐体を取り付ける">

筐体を取り付ける際に、ケーブルに引っ掛からないよう、コントローラのケーブルを引き出してから付けてください。

![筐体を取り付ける 1](/img/docs/taikoller/disassembly/step11-1.png)

![筐体を取り付ける 2](/img/docs/taikoller/disassembly/step11-2.png)

</Step>

</Steps>
