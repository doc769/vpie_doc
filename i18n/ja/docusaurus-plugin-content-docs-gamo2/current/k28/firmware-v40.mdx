# K28 ファームウェアアップデート V40 2022年6月28日

## 1 新機能

このファームウェアにはV39のすべての機能が含まれています。

1. キーボードレイアウト（B11、B12）で3.5mmオーディオジャックを使用する際に、ボタンが多重反応するバグを修復。
2. キーボードレイアウト（B11、B12）でのレポートレートを最適化し、1ms前後を保てるようになります。
3. B12レイアウトを増加。B14とB15が2つともSpaceボタンで、7鍵式ゲームに最適。
4. LED機能がアップグレード。

## 3 ファームウェア更新手順

### 3.1 HOST ファームウェア

1. K28のUSB-Cケーブルを抜いて、**B42＋B44**を押しながら、USB-Cケーブルを挿して、**2秒経ちましたら、B42とB44**を放します。
2. 圧縮ファイルを解凍し、**Update V1.1.exe**を開けて、**Connect**をクリックします。**Connect**が緑になりましたら、接続成功。成れなかった場合は、1を繰り返してください。
3. **Open files**ボタンを押して、ファームウェアファイル**HOST_xxxx.bin**を選択します。
4. **Update**ボタンをクリックします。
5. プロセスバーが100％になりましたら、**USB-Cケーブルを挿し直します**。
6. アップデート完了。

### 3.2 LED ファームウェア

1. K28のUSB-Cケーブルを抜いて、**B41＋B43**を押しながら、USB-Cケーブルを挿して、**5秒経ちましたら、B41とB43**を放します。
2. 圧縮ファイルを解凍し、**Update V1.1.exe**を開けて、**Connect**をクリックします。**Connect**が緑になりましたら、接続成功。成れなかった場合は、1を繰り返してください。
3. **Open files**ボタンを押して、ファームウェアファイル**LED_xxxx.bin**を選択します。
4. **Update**ボタンをクリックします。
5. プロセスバーが100％になりましたら、**USB-Cケーブルを挿し直します**。
6. アップデート完了。

## 4 Q&A

<FaqAccordion>
<FaqItem question="アップグレードの際に、何か注意点がありますか。">

キーコンビを押しながら、USBケーブルを挿してください。さしたら、2秒間か5秒間維持してから、ボタンを放してください。システムに認識されるには時間がかかります。そうしないと、コントローラーが認識されず、アップグレードできなくなります。

</FaqItem>
<FaqItem question="前のファームウェアをスキップして、直接このファームウェアでアップデートできますか。">

可能です。このファームウェアには前のファームウェアのすべての機能が含まれています。

</FaqItem>
<FaqItem question="アップグレード中にエラーが出たらどうしますか。">

USBケーブル接続不良で、データ交換に影響してしまうことが考えられますので、改めてアップグレードすれば大丈夫です。

</FaqItem>
<FaqItem question="このファームウェアが好きではないので、前のファームウェアに戻せますか。">

もちろん可能です。こちらをクリックして、ファームウェアをダウンロードしてください。20200221.zip

</FaqItem>
<FaqItem question="カスタマイズレイアウトはどうすればできますか。">

取扱説明書を確認してください。通常の手順は下記の通りです。①まずは紙に希望のレイアウトを描きます；②キーコンビでカスタマイズレイアウトに切り替えます；③ボタンを一つずつ設定します。④保存します。

</FaqItem>
<FaqItem question="どうしてConnectをクリックしたら、invalid IDの表示が出たのですか。">

**Config.ini**ファイルを削除しないでください。もしうっかりして削除してしまったら、ファームウェアの圧縮ファイルをダウンロードし直してください。

</FaqItem>
<FaqItem question="どうやってSteamでPlayStationレイアウトを使用しますか。">

もしゲーム内でPlayStationジョイスティックを使用可能でしたら、K28 Steamのレイアウトのいずれを使用してください。

</FaqItem>
</FaqAccordion>
