# K28　ファームウェアアップグレード　V39　2021年6月7日

## アップグレード内容

1. PlayStation4、Switch、PCでの送信アルゴリズムをより適切にアップグレード。
2. Switchのカスタムレイアウトを2つ増加。**B44+B17、B44+B18で設置可能。**
3. Switch1台には同時にK28　2台が接続可能。
4. Steamのカスタムレイアウトを2つ増加、PlayStation4ゲームパッドとして適用可能。**B44+B13、B44+B14で設置可能。**
   - カスタムレイアウトは元々ボタン配置がないので、まずはカスタマイズ機能でお好きのゲームに合わせて、ボタン配置を設定してください。
   - 設定画面で、PlayStation Configurationの支持をオンにしてください。
5. キーボードレイアウトでのポーリングレートは1msです。
6. IC過熱でポーリングレートが不安定になる状況を解決するため、LEDのアルゴリズムをより適切にアップグレード。
7. 2つのファームウェアバージョンを提供。
   - PlayStation4でのポーリングレートが1ms、Switchでのポーリングレートが1ms。（これは最大限であり、判定抜けの可能性があります。だ足し、それは可能性で、テスト中に判定抜けの状況はありませんでした。もし判定抜けの状況がありましたら、4msバージョンで更新してください。4msバージョンでも判定抜けの状況がありましたら、メールでご連絡ください）
   - PlayStation4でのポーリングレートが4ms、Switchでのポーリングレートが4ms。（安定）
8. オーディオに対する支持をアップグレード

## アップグレードガイド

1. K28のUSBーCケーブルを抜いて、**B42 + B44を押しながら**、USBーCケーブルをさしてください。2秒ほどして、**B42とB44を放ってください。**
2. ファームウェアの圧縮ファイルを通常ファイルに解凍して、Update V1.1.exeという名のファイルを開けてください。**Connectボタンをクリックして、もし緑色表示になったら、接続が成功し、次のステップに移ります。もし赤色表示になってら、接続が失敗し、1の操作を繰り返します。**
3. **Open files ボタンをクリックして、更新したいファームウェアバージョン（ファイル名はxxxx.bin）を選択します。**
4. **Update ボタンをクリックしてください。**
5. プログレスバーが最後まで進めたら、K28のUSBーCケーブルを抜き差ししてください。
6. アップグレード完了。

## Q&A

<FaqAccordion>
<FaqItem question="アップグレードの際に、何か注意点がありますか。">

キーコンビを押しながら、USBケーブルを挿してください。さしたら、2秒間維持してから、ボタンを放してください。システムに認識されるには時間がかかります。そうしないと、コントローラーが認識されず、アップグレードできなくなります。

</FaqItem>
<FaqItem question="前のファームウェアをスキップして、直接このファームウェアでアップデートできますか。">

可能です。このファームウェアには前のファームウェアのすべての機能が含まれています。

</FaqItem>
<FaqItem question="アップグレード中にエラーが出たらどうしますか。">

USBケーブル接続不良で、データ交換に影響してしまうことが考えられますので、改めてアップグレードすれば大丈夫です。

</FaqItem>
<FaqItem question="このファームウェアが好きではないので、前のファームウェアに戻せますか。">

もちろん可能です。こちらをクリックして、ファームウェアをダウンロードしてください。20200221.zip

</FaqItem>
<FaqItem question="カスタマイズレイアウトはどうすればできますか。">

取扱説明書を確認してください。通常の手順は下記の通りです。①まずは紙に希望のレイアウトを描きます；②キーコンビでカスタマイズレイアウトに切り替えます；③ボタンを一つずつ設定します。④保存します。

</FaqItem>
<FaqItem question="どうしてConnectをクリックしたら、invalid IDの表示が出たのですか。">

Config.iniというファイルを削除しないでください。もしうっかりして、削除したら、もう一度ダウンロードして、解凍すればいいです。

</FaqItem>
<FaqItem question="Steamで、どうやってPlayStation支持を適用しますか。">

もしそのゲームにPlayStationのゲームパッドが適用可能でしたら、K28のSteam レイアウトのいずれ選択してください。

</FaqItem>
</FaqAccordion>
