# GITALLER ファームウェアアップグレードチュートリアル

## ファームウェアのダウンロード

[ファームウェアをダウンロード](https://www.dj-dao.com/firmware/Guitar_firmware_20200926.zip)

:::tip
完全に解凍してからご使用ください。圧縮パッケージ内から直接実行しないでください。
:::

## ファームウェアアップグレード手順

<Steps>
<Step title="ファームウェアを解凍">

ダウンロードしたファームウェアの圧縮パッケージをローカルフォルダに解凍します。

</Step>
<Step title="USB が未接続であることを確認">

GITALLER の USB ポートが PC に接続されていないことを確認します。

</Step>
<Step title="ファームウェアアップグレードモードに入る">

START + SELECT ボタンを同時に押したまま、GITALLER の USB ポートを PC に挿入します。USB を PC に接続したら、START + SELECT ボタンを放してください。

![START + SELECT を押す](/img/docs/gitaller-firmware/step3-buttons.png)

![USB を挿入](/img/docs/gitaller-firmware/step3-usb.png)

</Step>
<Step title="ファームウェアアップグレードプログラムを開く">

`GITALLER_firmware_APP.exe` プログラムを開きます。

![プログラムを開く](/img/docs/gitaller-firmware/step4-app.png)

</Step>
<Step title="ファームウェアファイルを読み込む">

**LoadFile** ボタンをクリックし、ファームウェアをダウンロードしたフォルダーを見つけて、`GITALLER_firmware_V1.0_20200926.bin` ファイルを選択します。

![LoadFile をクリック](/img/docs/gitaller-firmware/step5-loadfile.png)

![ファームウェアファイルを選択](/img/docs/gitaller-firmware/step6-select-bin.png)

</Step>
<Step title="アップグレード開始">

**Start** ボタンをクリックして、ファームウェアのアップグレードを開始します。進行状況バーが完了するのを待ちます。

![Start をクリック](/img/docs/gitaller-firmware/step7-start.png)

![進行状況バーを待つ](/img/docs/gitaller-firmware/step8-progress.png)

</Step>
<Step title="アップグレード完了">

成功が表示されたら、**GITALLER USB インターフェースを再接続**します（重要）。

![アップグレード成功](/img/docs/gitaller-firmware/step9-success.png)

:::warning
成功しない場合は、ステップ 3 から操作を再度行ってください。
:::

</Step>
</Steps>

## ファームウェアのアップグレード後の機能について

### WAILING BUTTON モード

このモードはデフォルトであり、切り替える必要はありません。

B1+B2+B3+B4+B5+SELECT を 2 秒間同時に押して、ジャイロスコープの XYZ 軸出力をシールドし、wailing_up と wailing_down をボタンとして出力します。

![WAILING BUTTON モード](/img/docs/gitaller-firmware/wailing-button-mode.png)

### WAILING XYZ モード

このモードは上級者に適しています。初心者は切り替えないでください。

B1+B2+B3+B4+B5+START を 2 秒間同時に押して、wailing_up と wailing_down のボタン出力をシールドし、ジャイロスコープの XYZ 軸が正常に出力されます。

![WAILING XYZ モード](/img/docs/gitaller-firmware/wailing-xyz-mode.png)

## 解決した問題

1. GITADORA trial 1.0 では、START ボタンは開始機能であり、SELECT ボタンは HELP 機能です
2. DTXmania を設定するときは、WAILING BUTTON モードを使用してください。wailing_up ボタンと wailing_down ボタンの両方をマッピングする必要があります
3. S tools と B tools を使用する場合、XYZ 軸をマッピングする場合は WAILING XYZ モードに切り替えてください。wailing_up と wailing_down ボタンをマッピングする場合は WAILING BUTTON モードに切り替えてください

### GITADORA trial の設定図

GITADORA trial では、1 番目のボタンは 0 として表示され、wailing_up は 10 として表示され、wailing_down は 11 として表示されます。

![GITADORA trial 設定](/img/docs/gitaller-firmware/gitadora-trial-config.png)

### DTXmania の設定図

DTXmania では、1 番目のボタンは 1 と表示され、wailing_up は 11 と表示され、wailing_down は 12 と表示されます。

![DTXmania 設定](/img/docs/gitaller-firmware/dtxmania-config.png)
