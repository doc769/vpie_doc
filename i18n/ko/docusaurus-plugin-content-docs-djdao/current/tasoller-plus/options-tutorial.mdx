# TASOLLER Options 사용 가이드

## 1. TASOLLER Options 란?

- TASOLLER PLUS(2세대 제품)와 호환；TASOLLER(1세대 제품)와는 비호환；
- GroundSlider의 임계값(감도)을 조정；
- GroundSlider의 출력 지연을 수정；
- Air 및 GroundSlider의 조명 밝기를 조절；
- 여러 동작 모드 간 전환 활성화；
- 현재 터치 감지 값을 실시간으로 표시；

#### 3.1.1. 전체 임계값

오입력（과민감도로 인한 의도하지 않은 트리거）이 발생하는 경우, 임계값을 높여 오작동 가능성을 줄이는 것을 권장.

- 기본 권장값은 85；
- 입력 누락（감도 부족으로 인한 미감지）이 발생하는 경우, 임계값을 낮추는 것을 권장；
- 최적의 결과를 위해 감도 설정을 1~2 단위씩 조정하고 각 조정 후 테스트하는 것을 권장；

:::note
기준값은 [라오마오 글러브]로 테스트하여 얻은 것입니다. 맨손 감각을 기준으로 사용하세요.
:::

#### 3.1.2. BR 값（의도하지 않은 트리거 및 키 스택 관련）

- 20으로 설정；얇은 글러브를 사용하는 대부분의 플레이어에게 적합（얇은 글러브는 도전성 손가락으로 스마트폰 조작 가능）；
- 25로 설정；맨손 플레이어에게 적합；
- 7~10으로 설정；매우 가볍게 누르는 플레이어, 또는 두꺼운 글러브（마이마이 글러브 유사）를 사용하는 플레이어에게 적합；

:::note
값이 클수록 키 스택이 줄어든다. 키 누락 확률도 높아지므로 적절한 힘으로 누를 것.
절충값을 사용하는 것이 좋다. 대부분의 플레이어는 20 사용을 권장.
얇은 글러브의 기준은 [Old Cat Gloves].
:::

#### 3.1.3. Groundslider 지연

전제 조건：게임 내 B 매개변수（지연 매개변수）는 아케이드（일본 캐비넷 또는 인터내셔널 캐비넷）의 매개변수를 사용；

- 35±2로 설정, 사용 느낌에 따라 미세 조정；가정용 120Hz 디스플레이, 인터내셔널 캐비넷 60Hz에서 주로 플레이하는 경우 적합；
- 0~2로 설정, 초기값은 2 권장；가정용 120Hz 디스플레이, 일본 캐비넷 120Hz에서 주로 플레이하는 경우 적합；

:::note
지연 매개변수는 레이팅 17.4 두 명의 고수에 의해 테스트되어 강한 참고값을 가짐.
게임에서 Fast가 많은 경우, 지연을 증가시킴（+1 또는 +2）；
게임에서 Late가 많은 경우, 지연을 감소시킴（-1 또는 -2）；
:::

#### 3.1.4. Groundslider 조명 밝기

자신의 취향에 맞게 조정, 권장 범위는 70-100；

#### 3.1.5. AIR 조명 밝기

- 컴퓨터 USB2.0 Type-A 인터페이스인 경우, 밝기를 70 미만으로 조정 권장；
- 컴퓨터 또는 충전기 USB3.0 Type-A 인터페이스인 경우, 밝기를 자유롭게 조정 가능；
- CtoC 케이블인 경우, 밝기를 자유롭게 조정 가능（일반적으로 CtoC 케이블은 5V/3A 제공）；

#### 3.1.6. 게임 내 설정（중요）

- 아케이드에서의 자신의 A B 매개변수 사용；
  - Fast가 많은 경우, Options에서 지연 증가；
  - Late가 많은 경우, Options에서 지연 감소；

:::note
게임 TEST에서 임계값은 반드시 20으로 설정；
TASOLLER Options 소프트웨어는 게임과 공존 가능. 게임에서 데스크탑으로 전환하여 조정하고 게임으로 돌아올 수 있음.
:::

## 4. FAQ

<FaqAccordion>
<FaqItem question="FN1과 FN2의 기능은?">

- FN1 단일 클릭, TEST；
- FN2 단일 클릭, Service；
- FN2 더블 클릭, 녹색 진행 표시줄이 나타나며 키 스택 문제를 해결하는 빠른 Ground Slider 캘리브레이션 기능 사용（약 5초 소요）；
- FN2 트리플 클릭, 빨간 진행 표시줄이 나타나며 키 스택 문제를 해결하는 심층 Ground Slider 캘리브레이션 기능 사용（약 10초 소요）；

</FaqItem>

<FaqItem question="캘리브레이션 기능（매개변수 변경 없음, 게임 내 판정이 정상일 때도 있고 비정상일 때도 있음）">

- 게임 설정, 그래픽 카드 설정, 백그라운드 프로그램, Windows 자동 업그레이드 제외；
- FN2 더블 클릭, 약 5초 동안 녹색 진행 표시줄로 캘리브레이션；캘리브레이션 후 재시도；
- FN2 더블 클릭으로 해결되지 않으면 FN2 트리플 클릭 시도；
- FN2 트리플 클릭, 약 10초 동안 빨간 진행 표시줄로 캘리브레이션；캘리브레이션 후 재시도；

:::note
캘리브레이션 중 Ground Slider에 아무것도 올려놓지 말 것. 캘리브레이션에 영향을 줌.
FN2 기능은 게임을 종료할 필요 없음. 언제든지 사용 가능, 횟수 제한 없음；
:::

</FaqItem>

<FaqItem question="키가 스택된 경우 대처 방법（손을 떼도 키가 눌린 상태）">

- 글러브가 너무 젖지 않았는지 확인（먼저 맨손으로 키가 스택되는지 확인. 맨손으로 정상이면 건조한 글러브로 교체）；
- 마이마이 두꺼운 글러브 사용 비권장；
- 손상된 글러브 사용 비권장（손상된 글러브에는 많은 기름때가 포함되어 컨트롤러 동작에 영향）；
- 겨울에는 건조하여 정전기가 발생하기 쉬움. 컨트롤러 옆에 소형 가습기 배치 권장. 컨트롤러에서 1~1.5미터 거리. 너무 가까우면 [젖은 글러브]와 유사한 부작용 발생；
- 가습기가 없는 경우, 물티슈 또는 젖은 타올（사용 전 충분히 짜기）로 컨트롤러를 닦고, 티슈로 남은 수분 제거；

</FaqItem>

<FaqItem question="임계값과 감도의 관계는?">

- 임계값은 특정 힘으로 누를 때의 최소 트리거 조건을 말함.
- 임계값과 감도는 반비례 관계. 임계값이 클수록 감도가 낮음；임계값이 작을수록 감도가 높음；

</FaqItem>

<FaqItem question="게임 테스트의 임계값을 왜 20으로 설정하는가?">

게임 하드웨어에는 임계값 메커니즘이 있음. 20, 40, 60, 80. 이 조정 범위는 비교적 거칠므로 게임 테스트의 임계값은 조정하지 않고 20으로 고정하는 것을 권장.

</FaqItem>

<FaqItem question="Options의 고급 임계값 설정이란?">

이는 공식 기술 지원을 위해 예약된 디버깅 기능으로, 플레이어가 사용하는 것은 강력히 권장하지 않음.

</FaqItem>
</FaqAccordion>
