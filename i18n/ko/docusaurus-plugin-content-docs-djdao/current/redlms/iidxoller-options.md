# IIDXOLLER Options 소개

## 1. IIDXOLLER Options 소개

DAOdesign 팀은 다년간의 경험을 바탕으로 이 소프트웨어를 만들었습니다.

사용 중 문제가 발생하거나 다른 제안 사항이 있는 경우, 이메일 또는 트위터를 통해 문의해 주시면 적극적으로 개선하겠습니다.

다양한 모드가 미리 만들어져 있으며, 모드별로 다른 매개변수를 설정할 수 있어 편리하고 빠릅니다.

## 3. 펌웨어 설치

1. USB type-B 포트 1P와 2P를 동시에 연결해야 합니다
2. Open 버튼을 클릭합니다
3. 폴더에서 `SS900_RED-LMS_xxxxx.skystar` 펌웨어를 선택합니다
4. Update 버튼을 클릭합니다
5. 완료

:::warning
- 업그레이드 과정 중에는 반드시 USB 연결을 유지해야 합니다. 업그레이드 중에 USB type-B 연결이 끊어지면 메인 컨트롤 PCBA에 펌웨어를 다시 구울 수 없게 될 수 있습니다.
- 설치에 실패했다는 메시지가 표시되면, 다시 시도해 주십시오.
:::

## 4. 사용 가이드라인

기본 설정만으로도 대부분의 IIDX 게임에 적용할 수 있습니다.

### 4.3 공통 설정

#### 4.3.1 공통 - 버튼 설정

![버튼 설정](/img/docs/iidxoller-options/kr/KR-IIDXOLLER-Options-guide-30-B-button.png)

| 항목 | 설명 |
|------|------|
| **B1** | 마이크로스위치를 놓을 때 발생하는 채터링을 처리하는 시간입니다. 기본값인 3ms를 권장합니다. VX-01-1C22 또는 VX-01-1C23 마이크로스위치 사용 시 중복 입력이 발생하면 이 값을 높여 일부 개선할 수 있습니다 |
| **B2** | 마이크로스위치를 누를 때 발생하는 채터링을 처리하는 시간입니다. 기본값인 10ms를 권장합니다. VX-01-1C22 또는 VX-01-1C23 마이크로스위치 사용 시 중복 입력이 발생하면 이 값을 높여 일부 개선할 수 있습니다 |
| **B3** | 버튼을 놓은 후 버튼 조명이 꺼지는 데 걸리는 시간. 일반적으로 0-200ms 권장 |
| **B4** | 버튼 설정을 기본값으로 복원 |

:::tip
- B1+B2 < 50ms. 이것은 추정값이며 단일 버튼의 최대 입력 빈도와 관련이 있습니다. 일반적으로 단일 버튼의 초당 입력 횟수는 20회 미만입니다.
- VX-01-1C22 및 VX-01-1C23 사용 중 중복 입력이 발생하는 경우 B1+B2 시간을 늘리면 일부 증상이 완화될 수 있습니다. 중복 입력이 계속되면 마이크로스위치를 빠르게 교체하는 것이 좋습니다.
:::

#### 4.3.2 공통 - 턴테이블 설정

다소 복잡해 보이지만, 실제로 자주 조정하는 것은 B7과 B8입니다.

![턴테이블 설정](/img/docs/iidxoller-options/kr/KR-IIDXOLLER-Options-guide-31-B-TT.png)

| 항목 | 설명 |
|------|------|
| **B5** | 턴테이블 반전 시간 간격. EZ2ON 플레이 시에만 턴테이블을 2개의 버튼으로 설정하고 8ms 이상으로 설정해야 합니다. 기본 권장값: 0ms |
| **B6** | 권장값: 20~100ms. 값이 작을수록 턴테이블을 트리거하는 데 더 빠른 속도가 필요합니다. 기본 권장값: 100ms |
| **B7** | 감도로 이해할 수 있습니다. 값이 작을수록 감도가 높습니다. 기본 권장값: 6 (테스트 결과 6이 아케이드 느낌에 가장 가깝습니다). 스크래치 집중 플레이어는 2로 설정할 수 있습니다 |
| **B8** | 턴테이블 반전 감도로 이해할 수 있습니다. 기본 권장값: 4. 스크래치 집중 플레이어는 2로 설정할 수 있습니다 |
| **B9** | 1:2 사용 시 낮은 턴테이블 속도로 곡 선택 메뉴를 빠르게 스크롤할 수 있습니다. 기본 권장값: 1:4 |
| **B10** | 턴테이블 댐핑이 트리거되는 시간. 기본 권장값: 40ms |
| **B11** | 현재 턴테이블에 적용된 댐핑 레벨. F 영역의 매핑 값을 사용합니다 |
| **B12** | 댐핑 효과가 발생하기 전에 턴테이블이 회전해야 하는 각도를 나타냅니다. 2~16 사이의 다양한 값을 시도해 볼 수 있습니다 |
| **B13** | 턴테이블 설정을 기본값으로 복원 |

:::tip
- EZ2ON에서는 키보드 모드 또는 DS4 모드 사용을 권장합니다. 이 모드에서는 전역 매개변수를 사용하지 않고 개별적으로 설정할 수 있습니다: 턴테이블 반전 시간 간격 = 8ms.
- B6 = 10ms이고 턴테이블 댐핑이 높게 설정되면 같은 방향으로 여러 번 출력할 수 있습니다. 단, 부작용으로 느린 속도에서 턴테이블이 반응하지 않게 됩니다.
:::

### 4.4 미리보기 창

![미리보기 창](/img/docs/iidxoller-options/kr/KR-IIDXOLLER-Options-guide-40-C-preivew.png)

| 항목 | 설명 |
|------|------|
| **C1, C2** | 현재 PD 전원이 공급하는 전압을 표시합니다. 20V, 15V는 권장 작동 전압입니다. 12V는 장시간 사용 가능합니다. 9V는 작동하지만 장시간 사용은 권장하지 않습니다. 5V에서는 전자기 댐핑이 작동하지 않습니다. 0V는 PD 전원이 연결되지 않음을 의미합니다 |
| **C3, C4** | 턴테이블 데이터의 원시 값. 실제 출력에는 관여하지 않습니다 |
| **C5** | 현재 눌린 버튼을 표시합니다 |
| **C6** | 1P와 2P의 물리적 위치를 전환합니다. 메인 컨트롤 PCBA를 교체할 필요가 없는 한 이 버튼을 클릭하는 것은 권장하지 않습니다 |

:::tip
- 저희 경험에 따르면 Anker 140W PD 전원(Type-C 포트 3개, Type-A 포트 1개)은 두 개의 Type-C 케이블을 동시에 연결하면 동시에 20V 전압을 공급할 수 있습니다.
- 140W PD 전원 1개 또는 45W PD 전원 2개 사용을 권장합니다.
:::

#### 4.6.7 NS 모드

### 4.7 턴테이블 댐핑 매핑

턴테이블 무게는 공통 설정입니다. 취향에 따라 0-100 값을 자유롭게 입력할 수 있습니다.

![턴테이블 댐핑 매핑](/img/docs/iidxoller-options/kr/KR-IIDXOLLER-Options-guide-70-F-turntable-weight.png)

| 항목 | 설명 |
|------|------|
| **F1** | 댐핑 수치 입력 필드. 값이 클수록 댐핑이 커집니다 |
| **F2** | 클릭하여 현재 값을 저장. Enter 키를 눌러도 저장됩니다 |
| **F3** | 공장 기본값으로 복원 |

:::tip
- 일정한 댐핑 값을 원하는 경우 LEVEL1~LEVEL10을 모두 동일한 값으로 설정합니다.
- 댐핑 있음과 댐핑 없음을 혼합하려면 LEVEL1~LEVEL5를 0으로 설정하고 LEVEL6~LEVEL10을 동일한 값으로 설정합니다.
- 취향에 따라 자유롭게 설정하세요.
:::
