## 펌웨어 업그레이드 튜토리얼

<Steps>
<Step title="펌웨어 압축 파일 해제">

다운로드한 펌웨어 압축 파일을 로컬 폴더에 압축 해제합니다.

</Step>
<Step title="USB 미연결 확인">

GITALLER USB 케이블이 PC에 연결되어 있지 않은지 확인합니다.

</Step>
<Step title="펌웨어 업그레이드 모드 진입">

START + SELECT 버튼을 동시에 누른 채로, 다른 손으로 GITALLER USB 케이블을 PC에 연결합니다. USB가 연결되면 START + SELECT 버튼을 놓으세요.

![START + SELECT 누르기](/img/docs/gitaller/firmware-20201013/step3-buttons.png)

![USB 연결](/img/docs/gitaller/firmware-20201013/step3-usb.png)

</Step>
<Step title="펌웨어 업그레이드 프로그램 열기">

`GITALLER_firmware_APP.exe` 프로그램을 엽니다.

![프로그램 열기](/img/docs/gitaller/firmware-20201013/step4-app.png)

</Step>
<Step title="펌웨어 파일 로드">

**LoadFile** 버튼을 클릭하고, 다운로드한 펌웨어 폴더를 찾아 `GITALLER_firmware_V1.1_20201013_Sensitivity_14.bin`을 선택합니다.

Sensitivity_12는 최저 감도, Sensitivity_15는 최고 감도이며, Sensitivity_14 펌웨어를 권장합니다.

![LoadFile 클릭](/img/docs/gitaller/firmware-20201013/step5-loadfile.png)

![펌웨어 파일 선택](/img/docs/gitaller/firmware-20201013/step6-select-bin.png)

</Step>
<Step title="업그레이드 시작">

**Start** 버튼을 클릭하여 펌웨어 업그레이드를 시작합니다. 진행률 표시줄이 완료될 때까지 기다립니다.

![Start 클릭](/img/docs/gitaller/firmware-20201013/step7-start.png)

![진행률 표시줄 대기](/img/docs/gitaller/firmware-20201013/step8-progress.png)

</Step>
<Step title="업그레이드 완료">

성공 후, **GITALLER USB 케이블을 다시 연결합니다** (매우 중요).

![업그레이드 성공](/img/docs/gitaller/firmware-20201013/step9-success.png)

:::warning
성공하지 못한 경우 3단계부터 다시 시도하세요.
:::

</Step>
</Steps>

## 해결된 문제

1. 피킹 시 발생할 수 있는 다중 반응 문제 해결
2. wailing용 4가지 감도 펌웨어 제공

## 버튼 설정 참조

### GITADORA trial 설정

GITADORA trial에서 첫 번째 버튼은 0, wailing_up은 10, wailing_down은 11로 표시됩니다.

![GITADORA trial 설정](/img/docs/gitaller/firmware-20201013/gitadora-trial-config.png)

### DTXmania 설정

DTXmania에서 첫 번째 버튼은 1, wailing_up은 11, wailing_down은 12로 표시됩니다.

![DTXmania 설정](/img/docs/gitaller/firmware-20201013/dtxmania-config.png)
