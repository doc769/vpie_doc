## 펌웨어 업그레이드 방법

<Steps>
<Step title="펌웨어 압축 파일 해제">

다운로드한 펌웨어 압축 파일을 로컬 폴더에 해제합니다.

</Step>
<Step title="USB 연결 해제 확인">

GITALLER의 USB 포트가 PC에 연결되어 있지 않은지 확인합니다.

</Step>
<Step title="펌웨어 업그레이드 모드 진입">

START + SELECT 버튼을 동시에 누른 상태에서 GITALLER의 USB 포트를 PC에 연결합니다. USB를 PC에 연결한 후 START + SELECT 버튼을 놓으세요.

![START + SELECT 누르기](/img/docs/gitaller-firmware/step3-buttons.png)

![USB 연결](/img/docs/gitaller-firmware/step3-usb.png)

</Step>
<Step title="펌웨어 업그레이드 프로그램 실행">

`GITALLER_firmware_APP.exe` 프로그램을 실행합니다.

![프로그램 실행](/img/docs/gitaller-firmware/step4-app.png)

</Step>
<Step title="펌웨어 파일 로드">

**LoadFile** 버튼을 클릭하고, 펌웨어를 다운로드한 폴더를 찾아 `GITALLER_firmware_V1.0_20200926.bin` 파일을 선택합니다.

![LoadFile 클릭](/img/docs/gitaller-firmware/step5-loadfile.png)

![펌웨어 파일 선택](/img/docs/gitaller-firmware/step6-select-bin.png)

</Step>
<Step title="업그레이드 시작">

**Start** 버튼을 클릭하여 펌웨어 업그레이드를 시작합니다. 진행률 표시줄이 완료될 때까지 기다립니다.

![Start 클릭](/img/docs/gitaller-firmware/step7-start.png)

![진행률 대기](/img/docs/gitaller-firmware/step8-progress.png)

</Step>
<Step title="업그레이드 완료">

성공이 표시되면 **GITALLER USB 케이블을 다시 연결**합니다 (중요).

![업그레이드 성공](/img/docs/gitaller-firmware/step9-success.png)

:::warning
성공하지 못한 경우 3단계부터 다시 시도하세요.
:::

</Step>
</Steps>

## 펌웨어 업그레이드 후 기능 소개

### WAILING BUTTON 모드

이 모드는 기본 모드이며 전환할 필요가 없습니다.

B1+B2+B3+B4+B5+SELECT를 동시에 2초간 누르면 자이로스코프 XYZ축 출력이 차단되고, wailing_up과 wailing_down이 버튼으로 출력됩니다.

![WAILING BUTTON 모드](/img/docs/gitaller-firmware/wailing-button-mode.png)

### WAILING XYZ 모드

이 모드는 상급 플레이어에게 적합합니다. 초보 플레이어는 전환하지 마세요.

B1+B2+B3+B4+B5+START를 동시에 2초간 누르면 wailing_up과 wailing_down의 버튼 출력이 차단되고, 자이로스코프 XYZ축이 정상 출력됩니다.

![WAILING XYZ 모드](/img/docs/gitaller-firmware/wailing-xyz-mode.png)

## 해결된 문제

1. GITADORA trial 1.0에서 START 버튼은 시작 기능이고, SELECT 버튼은 HELP 기능입니다
2. DTXmania 설정 시 WAILING BUTTON 모드를 사용하세요. wailing_up과 wailing_down 두 버튼을 모두 매핑해야 합니다
3. S tools 및 B tools 사용 시 XYZ축을 매핑하려면 WAILING XYZ 모드로 전환하세요. wailing_up과 wailing_down 버튼을 매핑하려면 WAILING BUTTON 모드로 전환하세요

### GITADORA trial 설정 화면

GITADORA trial에서 첫 번째 버튼은 0으로 표시되고, wailing_up은 10으로 표시되며, wailing_down은 11로 표시됩니다.

![GITADORA trial 설정](/img/docs/gitaller-firmware/gitadora-trial-config.png)

### DTXmania 설정 화면

DTXmania에서 첫 번째 버튼은 1로 표시되고, wailing_up은 11로 표시되며, wailing_down은 12로 표시됩니다.

![DTXmania 설정](/img/docs/gitaller-firmware/dtxmania-config.png)
