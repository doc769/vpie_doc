# IIDXOLLER Options 소개

## 1. IIDXOLLER Options 소개

DAOdesign 팀은 다년간의 경험을 바탕으로 이 소프트웨어를 만들었습니다.

사용 중 문제가 발생하거나 다른 제안 사항이 있는 경우, 이메일 또는 트위터를 통해 문의해 주시면 적극적으로 개선하겠습니다.

다양한 모드가 미리 만들어져 있으며, 모드별로 다른 매개변수를 설정할 수 있어 편리하고 빠릅니다.

## 2. 다운로드

[IIDXOLLER Options V3.2](https://www.dj-dao.com/kr/support/23.html)

:::tip
반드시 압축을 완전히 해제한 후 사용하고, 압축 패키지 내에서 직접 실행하지 마십시오.
:::

## 3. 펌웨어 설치

1. USB type-B 포트 1P와 2P를 동시에 연결해야 합니다
2. Open 버튼을 클릭합니다
3. 폴더에서 `SS900_RED-LMS_xxxxx.skystar` 펌웨어를 선택합니다
4. Update 버튼을 클릭합니다
5. 완료

:::warning
- 업그레이드 과정 중에는 반드시 USB 연결을 유지해야 합니다. 업그레이드 중에 USB type-B 연결이 끊어지면 메인 컨트롤 PCBA에 펌웨어를 다시 구울 수 없게 될 수 있습니다.
- 설치에 실패했다는 메시지가 표시되면, 다시 시도해 주십시오.
:::

### 4.3 공통 설정

#### 4.3.1 공통 - 버튼 설정

![버튼 설정](/img/docs/iidxoller-options/kr/KR-IIDXOLLER-Options-guide-30-B-button.png)

| 항목 | 설명 |
|------|------|
| **B1** | 마이크로스위치를 놓을 때 발생하는 채터링을 처리하는 시간입니다. 기본값인 3ms를 권장합니다. VX-01-1C22 또는 VX-01-1C23 마이크로스위치 사용 시 중복 입력이 발생하면 이 값을 높여 일부 개선할 수 있습니다 |
| **B2** | 마이크로스위치를 누를 때 발생하는 채터링을 처리하는 시간입니다. 기본값인 10ms를 권장합니다. VX-01-1C22 또는 VX-01-1C23 마이크로스위치 사용 시 중복 입력이 발생하면 이 값을 높여 일부 개선할 수 있습니다 |
| **B3** | 버튼에서 손을 떼었을 때 버튼 조명이 꺼지는 데 걸리는 시간입니다. 보통 0-200ms 사이를 권장합니다 |
| **B4** | 버튼의 설정을 공장 초기값으로 복원합니다 |

:::tip
- B1+B2＜50ms. 이는 추정치이며, 하나의 버튼을 일정 시간 내에 누르는 최대 횟수와 관련이 있습니다. 일반적으로 단일 키 누름 속도는 초당 20회 미만입니다.
- VX-01-1C22 또는 VX-01-1C23에서 발생하는 중복 입력의 경우, B1+B2 시간을 늘리면 일부 증상이 완화될 수 있습니다. 만약 중복 입력이 계속 발생하면 마이크로스위치를 즉시 교체하는 것을 권장합니다.
:::

#### 4.3.2 공통 - 턴테이블 설정

조금 복잡해 보이지만, 실제로는 보통 B7과 B8만 조정하면 됩니다.

![턴테이블 설정](/img/docs/iidxoller-options/kr/KR-IIDXOLLER-Options-guide-31-B-TT.png)

| 항목 | 설명 |
|------|------|
| **B5** | 턴테이블의 방향을 반대로 돌릴때의 시간 간격입니다. EZ2ON에서만 턴테이블이 2개의 키로 설정됩니다. 이때는 간격을 8ms 이상으로 설정해야 합니다. 그렇지 않으면 EZ2ON의 고속 턴테이블에서 입력 손실이 발생할 수 있습니다. 권장 기본값은 0ms입니다 |
| **B6** | 권장 값은 20-100ms입니다. 값이 작을수록 턴테이블을 입력하는 데 더 빠른 속도가 필요하고, 값이 클수록 더 느린 속도가 필요합니다. 권장 기본값은 100ms입니다 |
| **B7** | 민감도로 이해할 수 있습니다. 값이 작을수록 민감도가 높습니다. 이 값은 턴테이블을 인식시키기 위해 턴테이블이 회전해야 하는 각도를 나타냅니다. 권장 기본값은 6입니다(테스트 결과 6이 아케이드 효과에 더 가깝다고 판단). 자신의 체감에 따라 세세하게 조정할 수 있습니다. 고난도 턴테이블 곡 플레이어는 2로 설정할 수 있습니다 |
| **B8** | 턴테이블을 반대로 돌릴 때의 민감도로 이해할 수 있습니다. 값이 작을수록 민감도가 높습니다. 이 값은 턴테이블을 반대로 입력하기 위해 턴테이블이 회전해야 하는 각도를 나타냅니다. 권장 기본값은 4입니다(테스트 결과 6이 아케이드 효과에 더 가깝다고 판단). 자신의 체감에 따라 미세 조정할 수 있습니다. 고난도 턴테이블 곡 플레이어는 2로 설정할 수 있습니다 |
| **B9** | 1:2 사용 시, 천천히 돌려도 곡 선택 메뉴를 빠르게 스크롤 할 수 있습니다. 1:4 사용 시, 더 빠르게 돌려야 곡 선택 메뉴를 빠르게 스크롤 할 수 있습니다. 권장 기본값은 1:4입니다(테스트 결과 1:4가 아케이드 효과에 더 가깝다고 판단) |
| **B10** | 턴테이블 장력이 작동하는 시간입니다. 값이 작으면 턴테이블을 천천히 돌릴 때 장력 효과가 없습니다. 임계 속도에서는 톱니바퀴처럼 걸리는 턴테이블 느낌이 나타납니다. 권장 기본값은 40ms입니다 |
| **B11** | 턴테이블의 현재 장력 LEVEL입니다. 장력 LEVEL의 구체적인 값은 F 구역의 값 설정을 사용합니다 |
| **B12** | 장력 효과가 발생하기 전에 턴테이블이 회전해야 하는 각도를 나타냅니다. 값이 클수록 장력 효과가 생기기 전에 더 큰 각도로 회전해야 합니다. 2~16 사이의 값으로 설정하여 실제 느낌을 확인해 볼 수 있습니다 |
| **B13** | 턴테이블 설정을 공장 기본값으로 복원합니다 |

:::tip
- EZ2ON에서는 키보드 모드 또는 DS4 모드를 사용하는 것이 좋습니다. 이 모드에서는 공통 설정을 사용하지 않고 턴테이블 반전 시간 간격을 8ms로 설정할 수 있습니다.
- B6=10ms이고 턴테이블 댐핑이 비교적 높게 설정된 경우, 같은 방향으로 여러 번 입력할 수 있습니다. 부작용으로는 턴테이블을 천천히 돌릴 때 반응하지 않을 수 있습니다.
:::

### 4.4 미리보기 창

![미리보기 창](/img/docs/iidxoller-options/kr/KR-IIDXOLLER-Options-guide-40-C-preivew.png)

| 항목 | 설명 |
|------|------|
| **C1、C2** | 현재 PD 전원 공급 장치에서 제공하는 전압을 표시합니다. 20V와 15V는 권장 작동 전압입니다. 12V에서는 장시간 사용할 수 있습니다. 9V에서는 작동은 하지만 장시간 사용은 권장하지 않습니다. 5V에서는 전자기 댐핑이 작동하지 않습니다. 0V는 PD 전원 공급 장치가 연결되지 않은 상태입니다 |
| **C3、C4** | 턴테이블 데이터의 원본 값이며 실제 입력에는 관여하지 않습니다 |
| **C5** | 현재 입력된 키를 표시합니다 |
| **C6** | 1P와 2P의 물리적 위치를 전환합니다. 메인 컨트롤 PCBA를 교체해야 하는 경우가 아니면 이 버튼을 클릭하지 않는 것이 좋습니다 |

:::tip
- 저희의 경험에 따르면, Anker의 140W PD 전원 공급 장치(Type-C 인터페이스 3개, type-A 인터페이스 1개)는 2개의 Type-C 케이블을 동시에 연결했을 때 20V 전압을 동시에 공급할 수 있습니다.
- 140W PD 전원 공급 장치 1개 또는 45W PD 전원 공급 장치 2개를 사용하는 것이 좋습니다.
:::

### 4.7 턴테이블 무게(장력) 설정

턴테이블 무게(장력)는 전역 설정입니다. 선호에 따라 0-100 사이의 값을 입력할 수 있습니다.

![턴테이블 무게 설정](/img/docs/iidxoller-options/kr/KR-IIDXOLLER-Options-guide-70-F-turntable-weight.png)

| 항목 | 설명 |
|------|------|
| **F1** | 장력 수치 입력 칸, 값이 클수록 장력이 커집니다 |
| **F2** | 버튼을 클릭하여 현재 값을 저장하거나, Enter 키를 눌러 현재 값을 저장합니다 |
| **F3** | 공장 기본값을 복원합니다 |

:::tip
- 일정한 장력 값을 사용하려면 LEVEL1~LEVEL10을 모두 같은 값으로 설정하면 됩니다.
- [장력 있음]과 [장력없음]만을 원한다면, LEVEL1~LEVEL5을 0으로, LEVEL6~LEVEL10을 똑같은 값으로 설정하면 됩니다.
- 원하는 대로 수치를 설정할 수 있습니다.
:::
