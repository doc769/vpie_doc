# K28 펌웨어 V39 20210607

## 1. 업그레이드 내용

1. PlayStation4, Switch, PC의 전송 알고리즘 최적화；
2. Switch 커스텀 레이아웃 2개 추가. 조합키 **B44+B17**, **B44+B18**으로 설정 화면 진입.
3. Switch 1대에 K28 2대 동시 연결 가능；
4. Steam에 커스텀 레이아웃 2개 추가, PlayStation DS4로 지원 가능. 조합키 **B44+B13**, **B44+B14**로 설정 화면 진입.
   - 커스텀 레이아웃은 초기 빈 상태이므로, 원하는 게임에 맞춰 모든 키 값을 직접 설정해야 합니다.
   - 설정 섹션에서 PlayStation DS4 Configuration 지원을 활성화해야 합니다.
5. **키보드 레이아웃**에서 보고율이 **1ms**로 변경됨；
6. IC 과열로 인한 간헐적 보고율 불안정 문제 해결을 위한 LED 알고리즘 최적화；
7. 펌웨어 2가지 버전 제공.
   - **PlayStation4 보고율 1ms**, **Switch 보고율 1ms** (이 속도는 콘솔의 한계로 키 누락이 발생할 수 있습니다. 가능성은 있으나 테스트 중에는 발견되지 않았습니다. 발견 시 4ms 버전 펌웨어를 사용하세요. 4ms 버전에서도 키 누락이 있으면 이메일로 문의하세요)
   - **PlayStation4 보고율 4ms, Switch 보고율 4ms** (안정적)
8. 오디오 지원 최적화.

## 3. 업그레이드 방법

1. K28의 USB를 분리하고 **B42+B44**를 동시에 누른 채로 USB를 연결합니다. **2초 후** B42+B44를 놓습니다.
2. 압축 파일을 해제합니다. **Update V1.1.exe** 파일을 열고 Connect 버튼을 클릭합니다. Connect 버튼이 **초록색**이 되면 K28 연결 성공입니다. 그렇지 않으면 1번 동작을 다시 시도하세요.
3. **Open files** 버튼을 클릭하고 대상 펌웨어 파일 **xxxx.bin**을 선택합니다.
4. **Update 버튼**을 클릭합니다.
5. 진행 바가 완료될 때까지 기다립니다. K28의 USB를 **재연결**합니다.
6. 완료.

## 4. Q&A

<FaqAccordion>
<FaqItem question="펌웨어를 플래시할 때 주의해야 할 사항은 무엇인가요?">

조합키를 누른 후 USB-C 케이블을 K28에 연결합니다. 약 2초 동안 키를 누른 상태를 유지한 후 놓습니다. Windows는 반응하는 데 시간이 필요합니다. 그렇지 않으면 K28 장치에 연결할 수 없습니다.

</FaqItem>
<FaqItem question="이전 펌웨어를 건너뛰고 이 펌웨어를 직접 업그레이드할 수 있나요?">

예. 이 펌웨어에는 이전 펌웨어의 모든 기능이 포함되어 있습니다.

</FaqItem>
<FaqItem question="펌웨어 플래시 중 오류가 발생하면 어떻게 해야 하나요?">

USB 인터페이스 접촉 불량으로 인해 데이터 전송 오류가 발생할 수 있습니다. 펌웨어를 다시 플래시하세요.

</FaqItem>
<FaqItem question="현재 펌웨어가 마음에 들지 않는데, 이전 버전으로 되돌릴 수 있나요?">

물론입니다. 펌웨어를 다운로드하려면 클릭하세요. 20200221.zip

</FaqItem>
<FaqItem question="나만의 레이아웃을 커스터마이즈하는 방법은?">

설명서를 주의 깊게 읽어주세요. 보통 ①먼저 원하는 레이아웃을 종이에 그립니다; ②조합키를 사용하여 커스터마이즈할 레이아웃 번호로 진입합니다; ③각 버튼의 키 값을 설정합니다; ④저장합니다.

</FaqItem>
<FaqItem question="Connect를 클릭하면 invalid ID가 표시되는 이유는?">

Config.ini 파일을 삭제하지 마세요. 실수로 삭제한 경우 펌웨어 업그레이드 패키지를 다시 다운로드하여 압축을 해제하세요.

</FaqItem>
</FaqAccordion>
