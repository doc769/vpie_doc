# K28 펌웨어 업데이트 V40 20220628

## 1 새로운 기능

이 펌웨어에는 펌웨어 V39의 모든 기능이 포함되어 있습니다.

1. 키보드 모드(B11, B12 레이아웃)에서 3.5mm 오디오 인터페이스 사용 시 버튼 다중 반응 버그 해결.
2. 키보드 모드(B11, B12 레이아웃)에서 유효 보고율 최적화, 약 1ms로 지속 유지 가능.
3. B12 레이아웃 추가. B14와 B15 키 모두 Space 키로, 7키 게임에 적합.
4. LED 조명 효과 최적화.

## 3 펌웨어 플래시 가이드

### 3.1 HOST 펌웨어

1. K28 USB-C 케이블을 분리하고 **B42 + B44**를 놓지 않은 채 누르고, USB-C 포트를 연결한 후 **2초 후 B42와 B44**를 놓습니다.
2. 펌웨어 압축 파일을 해제합니다. **Update V1.1.exe** 파일을 열고 Connect를 클릭합니다. Connect 버튼이 초록색으로 바뀌면 연결 성공입니다. 그렇지 않으면 1번을 반복합니다.
3. Open files 버튼을 클릭하고 업그레이드할 펌웨어 파일 "**HOST_xxxx.bin**"을 선택합니다.
4. Update 버튼을 클릭합니다.
5. 진행 바가 완료될 때까지 기다린 후 **K28 USB-C 케이블을 재연결**합니다.
6. 업데이트 완료.

### 3.2 LED 펌웨어

1. K28 USB-C 케이블을 분리하고 **B41 + B43**를 놓지 않은 채 누르고, USB-C 포트를 연결한 후 **5초 후 B41와 B43**를 놓습니다.
2. 펌웨어 압축 파일을 해제합니다. **Update V1.1.exe** 파일을 열고 Connect를 클릭합니다. Connect 버튼이 초록색으로 바뀌면 연결 성공입니다. 그렇지 않으면 1번을 반복합니다.
3. Open files 버튼을 클릭하고 업그레이드할 펌웨어 파일 "**LED_xxxx.bin**"을 선택합니다.
4. Update 버튼을 클릭합니다.
5. 진행 바가 완료될 때까지 기다린 후 **K28 USB-C 케이블을 재연결**합니다.
6. 업데이트 완료.

## 4 Q&A

<FaqAccordion>
<FaqItem question="펌웨어 업데이트 시 주의해야 할 사항은 무엇인가요?">

조합키를 누른 후 USB 포트를 연결합니다. USB를 연결한 후 약 2초 동안 유지하다가 조합키를 놓습니다. Windows는 반응하는 데 시간이 필요합니다. 그렇지 않으면 K28 장치에 연결할 수 없습니다.

</FaqItem>
<FaqItem question="이전 펌웨어를 건너뛰고 이 펌웨어를 직접 업그레이드할 수 있나요?">

물론입니다. 이 펌웨어에는 이전 펌웨어의 모든 기능이 포함되어 있습니다.

</FaqItem>
<FaqItem question="펌웨어 업데이트 중 오류가 발생하면 어떻게 해야 하나요?">

USB 인터페이스 접촉 불량으로 인해 데이터 전송 오류가 발생할 수 있습니다. 펌웨어를 다시 업데이트하세요.

</FaqItem>
<FaqItem question="현재 펌웨어가 마음에 들지 않는데, 이전 버전으로 되돌릴 수 있나요?">

물론입니다. 펌웨어를 다운로드하려면 클릭하세요. 20200221.zip

</FaqItem>
<FaqItem question="나만의 레이아웃을 커스터마이즈하는 방법은?">

설명서를 주의 깊게 읽어주세요. 보통 ①먼저 원하는 레이아웃을 종이에 그립니다; ②조합키를 사용하여 커스터마이즈할 레이아웃 번호로 진입합니다; ③각 버튼의 키 값을 설정합니다; ④저장합니다.

</FaqItem>
<FaqItem question="Connect를 클릭했을 때 invalid가 표시되는 이유는?">

**Config.ini** 파일을 삭제하지 마세요. 실수로 삭제된 경우 펌웨어 업그레이드 패키지를 다시 다운로드하여 압축을 해제하세요.

</FaqItem>
<FaqItem question="Steam에서 PlayStation 레이아웃을 어떻게 사용하나요?">

게임이 PlayStation 컨트롤러를 지원하는 경우, K28 Steam 레이아웃(B44+B13, B44+B14) 중 하나를 사용하세요.

</FaqItem>
</FaqAccordion>
