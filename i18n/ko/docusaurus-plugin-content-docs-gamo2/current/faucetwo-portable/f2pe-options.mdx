## 펌웨어 업데이트, 모드 전환, 전역 매개변수

![펌웨어 업데이트, 모드 전환, 전역 매개변수](https://support.gamo2.com/wp-content/uploads/2024/01/2024-01-21_23-47-33-1024x495.png)

| 번호 | 설명 |
|:---:|---|
| ① | 펌웨어 폴더 열기 |
| ② | 현재 펌웨어 버전 표시 |
| ③ | "업그레이드" 버튼 |
| ④ | 버튼 라이트가 천천히 꺼집니다. 기본값 600ms는 최대 밝기에서 0으로 600ms가 걸린다는 의미입니다. |
| ⑤ | Output HID 지원. Spixxtxxls 소프트웨어를 사용하는 경우 체크하는 것을 권장합니다. |
| ⑥ | 전역 라이트 밝기로 F2PE의 모든 라이트 밝기에 영향을 줍니다. |
| ⑦ | 키 디바운스 시간. 다중 키 트리거가 발생하면 값을 늘리십시오. 권장 범위: 6-15ms. 트리거 지연이 발생하지 않습니다. |
| ⑧ | 이 기능은 선곡 감도에만 영향을 미치며 게임 내 노브 성능에는 영향을 주지 않습니다. 기본값 사용을 권장합니다. 너무 빠르면 2를, 너무 느리면 3을 시도하세요. 1은 권장하지 않습니다. KSM 및 USC에는 거의 효과가 없습니다. |
| ⑨ | KD = 노브 디바운스. 기본값 사용을 권장합니다. 고속 직각 회전 시 조금 일찍 돌리십시오. 노브 출력이 불안정하면 4 또는 5를 시도하세요. |
| ⑩ | 언어 전환. 다른 언어를 사용하려면 `languagepack.xml`을 직접 수정하십시오. |
| ⑪ | 드롭다운 메뉴에서 모드를 선택하고 전환 버튼을 클릭하여 적용합니다. 기본값은 eAcloud Layout입니다. |

## 키보드 & 마우스 레이아웃

![키보드 & 마우스 레이아웃](https://support.gamo2.com/wp-content/uploads/2024/01/2024-01-21_23-48-39-1024x606.png)

| 번호 | 설명 |
|:---:|---|
| ① | "매핑" 버튼을 클릭하고 키보드의 원하는 키를 눌러 키 값을 바인딩합니다 |
| ② | 드롭다운 메뉴에서 마우스 X축, Y축을 선택합니다 |
| ③ | 현재 마우스 이동 방향을 반전시킵니다 |

## 키보드 레이아웃

![키보드 레이아웃](https://support.gamo2.com/wp-content/uploads/2024/01/2024-01-21_23-49-25-1024x716.png)

| 번호 | 설명 |
|:---:|---|
| ① | "매핑" 버튼을 클릭하고 키보드의 원하는 키를 눌러 키 값을 바인딩합니다 |
| ② | 매핑 버튼을 클릭하여 VOL-L의 시계 방향 및 반시계 방향 키 값을 설정합니다. 동일한 키 값으로 설정할 수 있습니다. |
| ③ | 매핑 버튼을 클릭하여 VOL-R의 시계 방향 및 반시계 방향 키 값을 설정합니다. 동일한 키 값으로 설정할 수 있습니다. |
| ④ | 시계 방향과 반시계 방향 전환 시 연장 시간. DJMAX RESPECT 및 EZ2ON에 유용합니다. 기본값 8ms를 권장합니다. |
| ⑤ | 노브가 멈출 때 키 값이 유지되는 시간. 필요에 따라 설정하십시오. |

## Steam 레이아웃

![Steam 레이아웃](https://support.gamo2.com/wp-content/uploads/2024/01/2024-01-21_23-50-13-1024x771.png)

| 번호 | 설명 |
|:---:|---|
| ① | 드롭다운 메뉴에서 DS4 키 값을 선택합니다. 동일한 키 값 선택을 지원합니다. 노브의 아날로그 값과 버튼 키 값은 상충 관계로 동일하게 설정할 수 없습니다. **예시:** *VOL-L을 LeftStick X-axis로 설정하면, 버튼을 LeftStick left 또는 LeftStick right로 설정할 수 없습니다.* |
| ② | 노브를 게임패드의 임의 축으로 지정합니다. |
| ③ | 노브 시계 방향 및 반시계 방향 각각의 키 값을 지정합니다. 동일한 키 값으로 설정할 수 있습니다. |
| ④ | 회전 방향 전환 시 지연 시간. DJMAX RESPECT 및 EZ2ON에 유효합니다. 1ms 사용 시 두 게임에서 입력이 누락될 수 있습니다. 8ms를 권장합니다(판정에 영향 없음). |
| ⑤ | 노브가 멈출 때 키 값이 유지되는 시간. 필요에 따라 설정하십시오. |

## 라이팅 설정

![라이팅 설정](https://support.gamo2.com/wp-content/uploads/2024/01/2024-01-21_23-50-46-1024x826.png)

| 번호 | 설명 |
|:---:|---|
| ① | 7개 버튼과 2개 노브 트리거 시의 라이트 색상을 설정합니다. RGB 색상의 밝기는 전역 라이트 밝기의 영향을 받습니다. |
| ② | BT 및 FX 텍스트의 라이트 색상. |
| ③ | 좌우 측면의 라이트 색상을 설정합니다. |
| ④ | 고정 색상 — 1가지 색상만 선택 가능합니다. |
| ⑤ | 그라데이션 색상 — 2가지 색상을 지정할 수 있습니다. 전환 색상은 알고리즘으로 자동 처리됩니다. |
