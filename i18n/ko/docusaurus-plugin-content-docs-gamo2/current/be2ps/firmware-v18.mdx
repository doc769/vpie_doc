## 3. 펌웨어 플래시 방법

<Steps>
<Step title="플래시 모드 진입">

BE2PS 버튼을 **누른 채로** 컴퓨터의 USB 포트에 연결합니다. **1초** 후 버튼을 놓습니다.

</Step>
<Step title="소프트웨어 연결">

펌웨어 압축 파일을 해제합니다. `Update V1.1.exe`를 열고 **Connect**를 클릭합니다.

**Connect가 녹색으로 바뀌면** 연결 성공입니다. 그렇지 않으면 1단계부터 반복합니다.

</Step>
<Step title="펌웨어 파일 선택">

**Open files** 버튼을 클릭하고 업그레이드할 펌웨어 파일 `xxxxx.bin`을 선택합니다.

</Step>
<Step title="업그레이드 시작">

**Update** 버튼을 클릭합니다.

</Step>
<Step title="완료 대기">

진행 표시줄이 완료될 때까지 기다립니다. 업그레이드가 완료됩니다.

</Step>
</Steps>
