## 3. 如何烧录固件

1. 必须同时连接 1P 和 2P 的 USB type-B
2. 点击打开固件按钮
3. 在文件夹中选择 `SS900_RED-LMS_xxxxx.skystar` 固件
4. 点击烧录固件按钮
5. 完成

:::warning
- 升级过程中，务必保持 USB 的连接。在升级过程中，如果出现 USB type-B 断开，可能会导致主控 PCBA 无法再次烧录固件。
- 如果提示烧录失败，请重新再次烧录。
:::

## 4. 使用指南

IIDXOLLER Options 提供了强大的参数，可以根据自己的喜好，进行定制。全默认的情况下，已经可以适配大部分 IIDX 游戏。

### 4.2 模式切换、睡眠、语言

![模式切换](/img/docs/iidxoller-options/cn/CN-IIDXOLLER-Options-guide-20-A-title.png)

| 项目 | 说明 |
|------|------|
| **A1** | 切换模式按钮，选择 A2 后，点击按钮生效 |
| **A2** | 模式列表 |

| **A4** | 2P 是否连接，如果未连接，显示灰色 |

### 4.3 全局功能设置

#### 4.3.1 全局 - 按键设置

![按键设置](/img/docs/iidxoller-options/cn/CN-IIDXOLLER-Options-guide-30-B-button.png)

| 项目 | 说明 |
|------|------|
| **B1** | 微动开关释放时，处理抖动的时间，建议使用默认值 3ms。如果 VX-01-1C22、VX-01-1C23 微动出现多重反应，适当增加这个值，会部分改善多重反应 |
| **B2** | 微动开关按下时，处理抖动的时间，建议使用默认值 10ms。如果 VX-01-1C22、VX-01-1C23 微动出现多重反应，适当增加这个值，会部分改善多重反应 |
| **B3** | 按键释放时，按键灯需要多少时间熄灭，通常建议在 0-200ms |
| **B4** | 恢复按键设置的出厂值 |

:::tip
- B1+B2＜50ms，此处是一个估算值。和单个按键的最高频率有关，通常单个按键的秒速小于 20次/秒。
- 对于 VX-01-1C22、VX-01-1C23 出现的多重反应，增加 B1+B2 的时间，可以缓解一些。如果依然出现多重反应，建议及时更换微动开关。
:::

#### 4.3.2 全局 - 转盘设置

看起来有一些复杂，实际上，常用是需要调节 B7 和 B8。

![转盘设置](/img/docs/iidxoller-options/cn/CN-IIDXOLLER-Options-guide-31-B-TT.png)

| 项目 | 说明 |
|------|------|
| **B5** | 转盘回拉的时间间隔。只有在 EZ2ON 时，转盘设置为 2 个键值，需要设置为 8ms 或者更大的值。否则，在 EZ2ON 高速转盘下，会出现丢失输入的情况。建议默认值 0ms |
| **B6** | 建议数值 20~100ms；数值越小，想要触发转盘，要求更快的速度；数值越大，想要触发转盘，可以使用更慢的速度。建议默认值 100ms |
| **B7** | 可以理解为灵敏度，数值越小，灵敏度越高。数值代表转盘输出值的时候，需要转动多大的角度。建议默认值 6（暂时我们测试，认为 6 比较接近街机效果），可以根据自己的体感进行微调。皿发狂的玩家可以设置为 2 |
| **B8** | 可以理解为转盘回拉的灵敏度，数值越小，灵敏度越高。数值代表转盘回拉输出值的时候，需要转动多大的角度。建议默认值 4（暂时我们测试，认为 6 比较接近街机效果），可以根据自己的体感进行微调。皿发狂的玩家可以设置为 2 |
| **B9** | 使用 1:2 时，较低的转盘速度可以触发选曲菜单的快速滚动；使用 1:4 时，需要使用较高的转速才可以触发选曲菜单的快速滚动。建议默认值 1:4（暂时我们测试，认为 1:4 比较接近街机效果） |
| **B10** | 转盘阻尼触发的时间。如果数值较小，在慢速转动转盘时，会出现没有阻尼的效果；如果在临界的转速下，会出现锯齿状的转盘手感。建议默认值 40ms |
| **B11** | 当前转盘调用的阻尼级别。具体的阻尼级别的值，使用的是 F 区的映射值 |
| **B12** | 代表转盘转动多少角度后，才会产生阻尼效果。数值越大，代表需要转动的角度越大才会产生阻尼效果。可以尝试不同的 2-16，试试感觉 |
| **B13** | 恢复转盘设置的出厂值 |

:::tip
- 在 EZ2ON 中，建议使用键盘模式或者 DS4 模式，可以在该模式下，不使用全局参数，单独设置：转盘反向时间间隔=8ms
- 如果将 B6=10ms，并且转盘阻尼设置比较大，可以实现相同方向多次输出。副作用是，慢速转动的时候，转盘没有响应
:::

### 4.4 预览窗口

![预览窗口](/img/docs/iidxoller-options/cn/CN-IIDXOLLER-Options-guide-40-C-preivew.png)

| 项目 | 说明 |
|------|------|
| **C1、C2** | 显示当前 PD 电源提供的电压。20V、15V 推荐的工作电压；12V 可以长期使用；9V 能工作，但不建议长时间使用；5V 电磁阻尼无法工作；0V 未插入 PD 电源 |
| **C3、C4** | 转盘数据的原始值，不参与实际的输出 |
| **C5** | 显示当前按下的按键 |
| **C6** | 切换 1P 和 2P 的物理位置。除非需要更换主控 PCBA，否则不建议点击这个按钮 |

:::tip
- 根据我们的经验，Anker 140W 的 PD 电源（具有 3 个 Type-C 接口，1 个 Type-A 接口），同时插入 2 个 Type-C 线缆时，可以同时提供 20V 电压。
- 建议使用 1 个 140W PD 电源；或者 2 个 45W PD 电源。
:::

### 4.6 模式

#### 4.6.1 PS2 and BLE 模式

在 PlayStation2 上使用，需要切换到这个模式，搭配 BE2PS 使用。

在 beatmania IIDX ULTIMATE MOBILE 上使用，需要切换到这个模式，搭配 BE2BT 使用。

![PS2 and BLE 模式](/img/docs/iidxoller-options/cn/CN-IIDXOLLER-Options-guide-60-E-PS2.png)

| 项目 | 说明 |
|------|------|
| **E1、E2** | 显示当前 1P 和 2P 转盘输出 X 轴的绝对值 |
| **E3、E4** | 1P 和 2P 转盘模拟在游戏中的输出 |
| **E5** | 在这个模式下，独立设置 1P 和 2P 的参数 |
| **E6** | 勾选代表使用全局设置；不勾选代表忽略全局设置，使用本模式下设置的参数 |
| **E7** | 设置 1P 和 2P 共同的参数 |
| **E8** | 恢复当前模式到默认值 |

### 4.7 转盘阻尼映射

转盘重量为全局设置。可以根据自己的感觉，任意输入 0-100 的数值。

![转盘阻尼映射](/img/docs/iidxoller-options/cn/CN-IIDXOLLER-Options-guide-70-F-turntable-weight.png)

| 项目 | 说明 |
|------|------|
| **F1** | 阻尼数字输入框，数值越大，阻尼越大 |
| **F2** | 点击按钮保存当前的数值；按下 Enter 也可以保存当前数值 |
| **F3** | 恢复出厂默认数值 |

:::tip
- 如果希望使用恒定的阻尼值，可以把 LEVEL1~LEVEL10 都设置为相同的数值
- 如果希望有阻尼值和没有阻尼值，可以将 LEVEL1~LEVEL5 都设置为零，LEVEL6~LEVEL10 设置为相同的数值
- 根据自己的喜好，可以任意设置
:::
