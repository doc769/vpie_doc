## 固件升级教程

<Steps>
<Step title="解压缩固件压缩包">

将下载的固件压缩包解压到本地文件夹。

</Step>
<Step title="确保 USB 未连接">

确保 GITALLER 的 USB 接口没有连接 PC。

</Step>
<Step title="进入固件升级模式">

同时按下 START + SELECT 按键不松手，将 GITALLER 的 USB 接口插入 PC。USB 插入 PC 后，START + SELECT 可以松开手。

![按下 START + SELECT](/img/docs/gitaller/firmware-20201013/step3-buttons.png)

![插入 USB](/img/docs/gitaller/firmware-20201013/step3-usb.png)

</Step>
<Step title="打开固件升级程序">

打开 `GITALLER_firmware_APP.exe` 程序。

![打开程序](/img/docs/gitaller/firmware-20201013/step4-app.png)

</Step>
<Step title="加载固件文件">

点击 **LoadFile** 按钮，找到下载固件的文件夹，选择 `GITALLER_firmware_V1.1_20201013_Sensitivity_14.bin` 文件。

固件后缀名字，Sensitivity_12 代表灵敏度最低，Sensitivity_15 代表灵敏度最高，推荐使用 Sensitivity_14 固件。

![点击 LoadFile](/img/docs/gitaller/firmware-20201013/step5-loadfile.png)

![选择固件文件](/img/docs/gitaller/firmware-20201013/step6-select-bin.png)

</Step>
<Step title="开始升级">

点击 **Start** 按钮，开始升级固件。等待进度条完成。

![点击 Start](/img/docs/gitaller/firmware-20201013/step7-start.png)

![等待进度条](/img/docs/gitaller/firmware-20201013/step8-progress.png)

</Step>
<Step title="升级完成">

显示成功后，**重新插拔 GITALLER USB 接口**（重要）。

![升级成功](/img/docs/gitaller/firmware-20201013/step9-success.png)

:::warning
如果没有成功，从步骤 3 开始重新再操作一遍。
:::

</Step>
</Steps>
